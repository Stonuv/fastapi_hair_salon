# Политика безопасности

## Поддерживаемые версии

Проект развивается на одной ветке (`main`), отдельных LTS-версий нет — актуальна
только последняя ревизия `main`.

## Как сообщить об уязвимости

**Не создавайте публичный GitHub Issue для уязвимостей** — это раскроет проблему
раньше, чем она будет исправлена.

Вместо этого используйте один из способов:

1. [GitHub Security Advisories](https://github.com/Stonuv/fastapi_hair_salon/security/advisories/new) —
   приватный канал сообщения, предпочтительный способ.
2. Email: stonuv@gmail.com.

Пожалуйста, укажите:
- Шаги для воспроизведения (или PoC).
- Затронутый компонент (backend/frontend/инфраструктура) и версию/коммит.
- Потенциальное влияние (какие данные/действия под угрозой).

## Что ожидать

Это учебный проект, поддерживаемый одним разработчиком в свободное время —
формального SLA нет, но подтверждение получения и первичная оценка обычно
занимают не больше нескольких дней. После исправления по согласованию с автором
отчёта возможна публикация деталей (например, через GitHub Security Advisory).
